The 'addDigits' esh plug-in
------------------------------

The 'addDigits' plug-in takes a number from the command adds up its digits and return the sum. It first turn the entered number 
into a long, then adds up the digits and return the sum. if the value entered is not a number, it will return 0.

Ex: addDigits 100
Sum of the digits is: 1


Ex: addDigits test
Sum of the digits is: 0
